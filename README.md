# Kakao Message Frontend

A web-based admin UI for managing Kakao message sending.

This frontend provides an interface to:
- Authenticate users via Kakao OAuth
- Select message recipients dynamically (all friends or predefined groups)
- Preview messages and send test messages
- Trigger message sending jobs
- View message delivery results (success / failure)

## Tech Stack

- Next.js
- TypeScript
- React
- (Optional) Chakra UI or Tailwind CSS
- Axios / Fetch API

## Features

- Kakao OAuth login flow
- Dynamic recipient selection (all / group-based)
- Message preview and test sending
- Message send execution
- Delivery result dashboard
